# MPU UI

MPU UI is a modern web interface for enterprise resource management, built with Next.js and TypeScript.

## Features

- Modern dashboard with real-time metrics visualization
- Interactive world map showing data center locations
- Resource management interface
- User management system
- Service monitoring
- Instance management
- Statistics and analytics

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- React Simple Maps
- Recharts
- Lucide Icons
- Shadcn/ui

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1. Clone the repository:
   \`\`\`bash
   git clone https://github.com/deadjoe/mpu_ui.git
   cd mpu_ui
   \`\`\`

2. Install dependencies:
   \`\`\`bash
   npm install

# or

yarn install
\`\`\`

3. Start the development server:
   \`\`\`bash
   npm run dev

# or

yarn dev
\`\`\`

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Development

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint
- `npm run format` - Format code with Prettier
